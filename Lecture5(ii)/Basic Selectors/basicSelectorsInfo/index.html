Basic Selctors basically 5 prakar ke hote h 1. Element Selectors 2. ID Selectors 3. Class Selectors 4. Attribute Selectors 5. Universal Selectors Element selectors = hum seeda tag name likh ke , curly braces laga ke use apni css likh dete h. ID selectors= giving any tag a Id which should be unique and apply changes on it is called id selectors Class selectors= giving class name to any tag, it can be assigned to any number of tags , changes are made will reflect to every class having same class name. Attribute selector= tag ka naam likh ke firr attribute ka naam likhna h then apna css apply karo. Universal selector= select all the tags in the document and then apply css to all the tags in the document.
